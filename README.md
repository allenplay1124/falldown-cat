# 接貓咪遊戲

一個簡單有趣的 HTML5 貓咪接接樂遊戲，玩家控制盤子接住從上方掉落的貓咪，挑戰你的反應速度！

## 遊戲規則

- 遊戲時間 **60 秒**，在時間內盡量接住更多貓咪
- 貓咪分為三種，分數越高體型越小：
  - **大貓咪** — 2 分（80px）
  - **中貓咪** — 5 分（60px）
  - **小貓咪** — 10 分（40px）
- 掉落速度會隨時間逐漸加快，挑戰性越來越高
- 時間結束後顯示總得分

## 操作方式

| 裝置 | 操作 |
|------|------|
| 桌機 | 方向鍵 ← →（或 A / D） |
| 桌機 | 滑鼠在遊戲區左右拖動 |
| 行動裝置 | 點擊下方 ← → 按鈕 |
| 行動裝置 | 在遊戲區左右滑動 |

## 技術特色

- 純前端開發，無框架依賴
- 支援桌機與行動裝置（RWD）
- 貓咪使用 PNG 圖片，依大小等比縮放
- 使用 Cubic 11 像素字體，風格一致
- 遊戲內無廣告或追蹤程式

## 技術架構

- **語言**：HTML5, CSS3, JavaScript (ES6+)
- **圖片**：PNG 格式，置於 `images/` 目錄
- **字體**：Cubic 11（woff2/woff），置於 `fonts/` 目錄
- **盤子**：CSS 繪製

## 啟動方式

直接開啟 `index.html` 即可遊玩，無需任何伺服器或建置工具。

## 瀏覽器支援

現代瀏覽器（Chrome、Firefox、Safari、Edge）皆可正常運行。
