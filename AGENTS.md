# AGENTS.md

## 專案概述
HTML5 接貓咪遊戲，純前端開發，無框架依賴。遊戲時間 60 秒，玩家控制下方盤子接住從上方掉落的貓咪。

## 遊戲機制
- **貓咪種類**：大貓咪（2 分）、中貓咪（5 分）、小貓咪（10 分）
- **掉落方式**：貓咪隨機從上方任一位置掉落
- **控制方式**：桌機使用方向鍵，行動裝置使用下方左右按鈕
- **掉落速度**：初始較慢，隨遊玩時間逐漸加快
- **結束條件**：遊戲時間 60 秒後顯示總得分

## 技術架構
- **語言**：HTML5, CSS3, JavaScript (ES6+)
- **資源管理**：貓咪圖片置於 `images/`，字體檔案置於 `fonts/`
- **RWD 支援**：桌機與行動裝置皆支援
- **CSS 繪製**：盤子以 CSS 技術實現；貓咪使用 PNG 圖片並依大小縮放

## 檔案結構
```
/
├── AGENTS.md              # 專案指引
├── index.html             # HTML5 遊戲主結構
├── style.css              # RWD 樣式與 CSS 繪圖
├── script.js              # 遊戲邏輯
├── images/                # 貓咪圖片
│   ├── big-cat.png
│   ├── mid-cat.png
│   └── small-cat.png
└── fonts/                 # 自訂字體
    ├── Cubic_11.woff2
    └── Cubic_11.woff
```

## 開發流程
1. 開啟 `index.html` 即可啟動遊戲
2. 桌機使用方向鍵控制盤子左右移動
3. 行動裝置點擊下方左右按鈕控制盤子
4. 遊戲結束後總分會顯示於畫面上

## 設計決策
- **圖片資源**：貓咪使用 PNG 圖片，依大小等比縮放（大 80px / 中 60px / 小 40px）
- **自訂字體**：使用 Cubic 11 像素字體，提升遊戲風格一致性
- **純前端**：無後端服務，純瀏覽器遊戲
- **繁體中文**：所有 UI 與文件使用繁體中文
- **漸增速度**：增加遊戲挑戰度，初期較容易上手

## 遊戲狀態
1. **開始畫面**：顯示遊戲說明及開始按鈕
2. **進行中**：貓咪掉落，盤子可控制
3. **結束畫面**：顯示總得分及重新開始選項

## 注意事項
- 所有 UI 與文件皆使用繁體中文
- 貓咪圖片使用 PNG 格式，置於 `images/` 目錄
- 字體使用 Cubic 11（woff2/woff），置於 `fonts/` 目錄
- 支援桌機與行動裝置
- 遊戲內無廣告或追蹤程式
